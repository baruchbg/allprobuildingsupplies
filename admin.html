Settings
Configure your GitHub connection and admin password.
To connect GitHub: You need a Personal Access Token with repo permissions. Go to GitHub → Settings → Developer Settings → Personal Access Tokens → Tokens (classic) → Generate new token → check repo → copy the token. Your repo name is the one where your site files live (e.g. yourusername.github.io).

Help
How to use the admin panel.
First Time Setup
Go to Settings and enter your GitHub username, repo name, branch (usually main), and Personal Access Token. Then come back to Products and click Load from GitHub.
Editing Products
Click any cell to edit it directly. Change prices, descriptions, sizes, quantities — anything. Edited rows highlight in gold. New rows highlight in green.
Adding a Product
Click + Add Product to add a blank row at the bottom. Fill in all the fields. Use the same Code as existing rows to group a product under the same category heading on the public site (e.g. multiple sizes of PVC-1/4HH).
Publishing Changes
Click Publish to Site in the top bar. This saves your changes to products.csv AND rebuilds products.html in your GitHub repo. Your live site updates within 1–2 minutes.
Deleting a Product
Click the ✕ button on the right side of any row to delete it. You'll be asked to confirm.
Security Note
This admin panel stores your GitHub token in your browser's local storage. Only use it on your own trusted device. Never share the admin.html URL or your password.
