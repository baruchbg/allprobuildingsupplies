Help
Editing Products
Click any cell to edit. Change prices, descriptions, quantities. Click Publish to Site when done — live in ~1 minute.
Adding a Product
Click + Add Product. Use the same Code as existing rows to group sizes together (e.g. PVC-1/4HH).
Approving Users
Go to the Users tab. New registrations show as "Pending". Click Approve to grant pricing access.
Publishing
Publish to Site saves products.csv AND rebuilds products.html on GitHub in one step.
