```

contact.html     → Contact form + info
assets/
  logo.png       → Logo (white + gold, transparent background)

  main.js        → Shared JavaScript (cursor, animations, nav)
```

## How to Deploy on GitHub Pages

1. Upload all these files to your GitHub repository root
2. Make sure GitHub Pages is set to deploy from the **main** branch, **root folder**
3. Visit your domain — changes go live in 1–2 minutes
